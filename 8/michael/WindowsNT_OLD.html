Windows NT

.

Windows NT is a family of Windows operating systems, that contains all the well-known Microsoft OS-s that are out there. NT was developed as Microsoft stopped partnering with IBM to create new operating systems in 1990s. It was created separately from other Windows families (e.g. Windows 3.X), and was planned to be a good OS for workstations (Windows NT Workstation) and servers (Windows NT Server). The NT gave most of the following Micrososft OS-s their base.
