Windows Longhorn

Windows Longhorn was the codename for Windows Vista. Despite the actual OS being the early Vista, it is often considered a completely separate OS, and for a reason: despite its successor's visual modernity, Longhorn was (mostly) a modified version of Windows XP, with minor changes such as colors, addition of a sidebar (that remained in Vista) and some framework tweaks.

Longhorn was developed in milestones. Each milestone was a massive step of development for the OS.

Early development

During this era of development, OS looked just like Windows XP, but already had improvements that weren't seen by a casual user. But this era also introduced the sidebar into OS, which sometimes caused BSOD, although everything was fixed later.

Early development 2

A version of Longhorn of this milestone was actually shown to the public, not leaked. It was shown on the WinHEC conference, although only screenshots remain of this conference. No OS was actually given for the public use. It showed some minor graphical improvements, as well as WinFS. WinFS was a system improvement that, for instance, allowed people to see their file directories' icons getting "filled" just like the recycle bin. Despite the fact that this was quite revolutionary for the time, of course, it caused BSOD.

Milestone 2

Build 3663 of 2002 is the first ACTUAL verion of Longhorn, that was somewhat separate from its XP base. It had a "luna" theme in it (although it was also included in Windows XP), but didn't really had anything that could separate its look from Windows XP (except sidebar). It was not leaked to the internet, only screenshots remain.

Milestone 3

Build 3683 of 2002 was leaked on October 20th, 2002. It was the first Longhorn build to be leaked to the public. Builds 3706-3718 also included in the "Milestone 3" era, include actual visual changes, logo was changed to transparent-white, Internet Explorer 6.05, theme changes, working WinFS and overall stability fixes. Build 3713 of 2002 was leaked in 2011, which is the longest time for all Milestone 3 Longhorn builds.

Milestone 4

Build 4001 of December 2002 was the first Milestone 4 Longhorn build to exist. It was one of the first builds to implement WIM based installation. it was the first major NT codebase. During installation it saves all OS files in one INSTALL.WIM disk file. When everything is loaded into WinPE, it is translated to the UI itself. When the WIM file is copied into a system disk partition, the system is then reloaded to finish OS installation. This was made to allow users to install OS in just 15 minutes.

Milestone 5

Builds 4015-4029 of 2003 include WIndows Explorer tweaks, desktop effects improvements, tooltips were added. Nothing that global because Microsoft were trying to release latest builds of this milestone as first public beta, but this never happened.

Milestone 6

Builds 4033-4040 of 2003 were the first builds to have "Aero glass" window view inside them. Aero glass was the very first prototype of Windows Aero ever released. They also had tweaks regarding other themes in this milestone of Longhorn.

Milestone 7

Builds 4042-4093 of 20003 and 2004 were the last versions of Windows Longhorn. They were improving many things regarding the system, BitLocker was introduced, WinFS was introduced, etc. But then Microsoft understood that making a Windows XP revamp was not a viable option for them, and so they decided otherwise.

Development reset

The whole development was reset. From 2004 to 2006 Microsoft was using the experience they gained, to create a more modern system under codename Longhorn. Eventually they released Windows Vista.
